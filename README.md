# 靶场项目

本项目用于构建一组**简单、可复现、可部署**的安全靶场环境，  
旨在为学习者提供一个**实践型**的练习平台，用于理解和体验常见的安全问题。

所有靶场均保持设计简洁，避免复杂依赖，强调“行为观察”和“结果反馈”。

---

## 项目目标

- 提供 **可直接使用的安全靶场**
- 降低学习与部署成本
- 通过真实交互方式理解安全问题
- 不以教学页面形式直接暴露漏洞原理
